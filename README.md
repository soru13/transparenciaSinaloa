# Empieza a desarrolloar:

### Ambientación: crea el archivo .env dentro ingresa:

```bash
ENV=dev
```

### **bold**, `Con Docker`.

```bash
docker-compose up -d dev
```

###para producción
primero el siguiente comando para que genere la carpeta Dist

```bash
docker-compose up -d dev
```

```bash
docker-compose up -d pro
```

### \*\*bold\*\*, `Sin Docker`.

1. npm i
2. npm run dev
